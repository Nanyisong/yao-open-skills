# Reference Materials

This directory contains the public source documents that informed the design of `yao-expert-skill`.

These files are background reading materials. They are not required at runtime. The executable skill contract remains in `SKILL.md`, `references/`, `templates/`, and `scripts/`.

## Files

| File | Format | Purpose |
|---|---|---|
| [industry_expert_skill_design.docx](industry_expert_skill_design.docx) | Word | Initial design plan for the expert-learning skill, including expected input/output shape and report structure. |
| [industry_learning_deep_research_report.docx](industry_learning_deep_research_report.docx) | Word | Deep research report for industry-learning methodology, including the structural logic behind expert reports and tutorials. |

## Suggested Reading Order

1. Read `industry_learning_deep_research_report.docx` to understand the research and learning logic.
2. Read `industry_expert_skill_design.docx` to see how the method was translated into a reusable Skill design.
3. Then read [`../SKILL.md`](../SKILL.md) and [`../references/report-and-tutorial-contract.md`](../references/report-and-tutorial-contract.md) to see the production implementation.
