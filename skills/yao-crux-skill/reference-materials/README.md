# 参考资料下载

这个目录用于存放 `yao-crux-skill` 的公开参考资料，方便读者理解这个 Skill 背后的理论来源、方法论设计和产品化思路。

这些资料是补充阅读材料，不是 Skill 运行时必须读取的规则文件。Skill 的执行规则仍以 `SKILL.md`、`references/`、`scripts/` 和 `templates/` 为准。

| 文件 | 格式 | 用途 |
|---|---|---|
| [复杂问题主要矛盾识别方法论调研报告.docx](复杂问题主要矛盾识别方法论调研报告.docx) | Word | 系统整理复杂问题中识别主要矛盾的方法论、科学思想、工具和 AI 产品化路径。 |
| [复杂问题主要矛盾识别方法论调研报告.html](复杂问题主要矛盾识别方法论调研报告.html) | HTML | 上述调研报告的网页阅读版，便于在浏览器中直接查看。 |

## 建议阅读顺序

1. 先读 `复杂问题主要矛盾识别方法论调研报告.html`，快速理解整体方法论。
2. 再读 `矛盾论底层原理与应用场景研究.docx`，补齐理论基础和场景迁移。
3. 最后读 `矛盾论AI_Skill设计方案.docx`，了解如何把方法论转成可复用的 AI Skill。
